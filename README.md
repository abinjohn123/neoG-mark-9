# top3Tech - Tech Recommendation App

A react app that recommends the best three tech products from four categories.

### Categories Available

1. Action Cameras
2. Powerbanks
3. Earphones
4. Smart Watches

## Tech Stack

1. HTML
2. CSS
3. JavaScript
4. React
